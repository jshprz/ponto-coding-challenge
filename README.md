# ponto-coding-challenge

Setting up the application

- npm install
- create ".env" file in the root app folder

".env" file should contain these variables:

APP_HOSTNAME=,
APP_PORT=,

POSTGRES_USER=,
POSTGRES_HOST=,
POSTGRES_DATABASE=,
POSTGRES_PASSWORD=,
POSTGRES_PORT=,

Note: Use higher version of Chrome for Web Speech Recognition
